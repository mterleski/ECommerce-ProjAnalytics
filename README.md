📌 Descrição
Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) em um conjunto de dados de vendas de e-commerce. A análise busca responder perguntas estratégicas sobre o comportamento de compra, produtos mais vendidos, desempenho ao longo do tempo, e padrões de clientes — com o foco final em gerar insights de negócio que possam ser aplicados na tomada de decisão.

📁 Fonte dos Dados

-O conjunto de dados contém informações detalhadas sobre transações, como:
-Identificação do pedido
-Tipo e categoria do produto
-Preço e quantidade
-Data da compra
-Informações pessoais dos clientes.

🧠 Objetivos do Projeto

-Analisar o comportamento de vendas ao longo do tempo
-Identificar produtos mais vendidos e com maior faturamento
-Entender o ticket médio e a frequência de pedidos
-Explorar a distribuição de vendas por categoria e tipo de produto
-Gerar insights visuais para tomada de decisão

🔧 Ferramentas Utilizadas
Python 3.9+
Pandas – manipulação e agregação de dados
sqlite3 – manipulação dos dados usando SQL

📊 Principais Análises Realizadas
📈 Evolução mensal das vendas
🛍️ Top produtos por volume de vendas e faturamento
💸 Análise de ticket médio por pedido
🧮 Quantidade total de pedidos e itens vendidos
🧩 Segmentação por categoria e tipo de produto
📅 Identificação de padrões sazonais

✅ Conclusões
Este projeto mostra como a análise de dados pode transformar informações brutas em insights valiosos para a estratégia de e-commerce. As técnicas e visualizações aplicadas podem ser adaptadas para ambientes reais de negócio.
